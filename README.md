# Raizes do Amanhã: Educação, cidadania e oportunidades para crianças e adolescentes
## Subtema
Tratando do subtema _Desigualdade Social_, o projeto _Raízes do Amanhã_ tem como objetivo apoiar o acesso a educação para crianças e adolescentes em situação de vulnerabilidade social. Ele busca combatê-la.

## Relação com geografia
A desigualdade social constitui uma das principais características do espaço geográfico urbano brasileiro. Segundo a Geografia, o espaço urbano não é produzido de forma homogênea, mas sim como resultado das relações econômicas, políticas e sociais que organizam a cidade e definem onde se concentram os investimentos, os serviços e as oportunidades.

Nesse contexto, observa-se que as áreas centrais e mais valorizadas das cidades apresentam maior oferta de infraestrutura urbana, enquanto as regiões periféricas e marginalizadas são marcadas pela precariedade dos serviços públicos, pela distância dos equipamentos urbanos e pela menor presença do Estado. Essa dinâmica configura um processo conhecido como segregação socioespacial, no qual grupos sociais são distribuídos de maneira desigual pelo território.

A educação, compreendida como um direito social fundamental e como um elemento estruturante do desenvolvimento humano, também é afetada por essa lógica espacial. Em muitas cidades, a localização das escolas, a qualidade da infraestrutura educacional e o acesso a projetos pedagógicos complementares variam significativamente entre os bairros, refletindo as desigualdades presentes no espaço urbano.

## Objetivo do projeto
Nesse contexto, o projeto busca não só combater a desigualdade social, como também entender a forma como ela afeta a qualidade do aprendizado de crianças e adolescentes em bairros periféricos, avaliando a infraestrutura das escolas e o desenvolvimento dos participantes.

Para isso, o projeto contará com um sistema web, responsável por coletar as informações dos participantes e organizá-las em um dashboard, tendo uma área dedicada para os administradores cadastrarem os participantes e organizar as informações de cada um.
## Escopo do projeto
### O que o projeto inclui
- Sistema web com página pública e área administrativa
- Autenticação de administradores por login e senha
- Cadastro de novos administradores com aprovação por administrador existente
- Cadastro de crianças e adolescentes participantes do projeto
- Registro de dados pessoais e educacionais por meio de formulários
- Sistema de doações ou pagamentos
- Edição e exclusão dos registros cadastrados
- Armazenamento dos dados em banco de dados
- Dashboard com visualizações gráficas dos dados coletados
- Visualização geográfica simplificada
- Página pública com apresentação do projeto e acesso às visualizações do dashboard
### O que o projeto não inclui
- Cadastro ou autenticação de visitantes
- Comunicação entre usuários (chat, mensagens ou fóruns)
- Agendamento de aulas ou gestão pedagógica
- Integração automática com bases de dados externas
- Mapas interativos complexos (Google Maps, OpenStreetMap, etc.)
- Aplicativo mobile ou desktop
- Coleta automática de dados educacionais
- Uso de dados sensíveis além dos necessários ao projeto
